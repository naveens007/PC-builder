PC Builds for budget hackers!

Desktop 📺: Surfing, email, office apps, music/video playback.

This formula is weighted towards single-core CPU performance:

50%*DCPU Bench + 10%*GPU Bench + 30%*SSD Bench + 10%*HDD Bench

(DCPU Bench = 80% Single core, 10% Quad core, 10% Multi core)


Gaming 🎮: 3D Gaming and graphics.

This formula is weighted towards GPU performance:

25%*GPU Bench + 15%*SSD Bench + 10%*HDD Bench

(GCPU Bench = 30% Single core, 60% Quad core, 10% Multi core)

For games that can't use multiple cores the desktop CPU Bench is a better indicator of performance.


Workstation 🖥: Number crunching, virtual machines, databases, audio/video encoding.

This formula is weighted towards multi-core CPU performance:

40%*WCPU Bench + 20%*GPU Bench + 25%*HDD Bench

(WCPU Bench = 10% Single core, 10% Quad core, 80% Multi core)


Note: Some of these are builds in which we've use an old 950/960 EVO SSD lying around instead of a new 970 EVO SSD!

Note: Some of these are builds use a G.SKILL Ripjaws series 3200 MHz CL16 DDR4 RAM. A new version of the same DDR4 RAM with CL14 has recently come.

Note: AMD says that the 2 ndgeneration Ryzen has a maximum memory bandwidth of 2933 Mhz.Some of our builds use RAM of higher bandwidth.
